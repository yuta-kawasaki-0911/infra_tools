## 本領域の説明
* ドメイン・SSL証明書の有効期限チェックを行うツールを管理する領域。

## 前提条件
* 特に無し。

## ツール説明
* DomainSSLChecker.sh
	* ドメイン・SSL証明書の有効期限チェックを行うツール。
	* 任意のサーバのcron等に登録することで、定期的に確認することが可能。
	* `VAR_Mail`に結果を通知したいメールアドレスを指定する。
* Domain_pid_list.txt
	* 有効期限をチェックしたいドメインを記載する。
	* 1行に1つドメインを定義、複数行定義することが可能。
* SSL_pid_list.txt
	* 有効期限をチェックしたいドメインを記載する。
	* 1行に1つドメインを定義、複数行定義することが可能。
